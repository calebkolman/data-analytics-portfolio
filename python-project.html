About This Project

This project leverages Python for analyzing flagged content updates among students. Using libraries like Pandas, Matplotlib, and Seaborn, we uncover trends and insights in user behavior.

Data Visualization with Summary

Bar Graph 1

The chart below shows the timeframes in which students returned to update their flagged content after the initial scan:

  • Same Day Updates: The majority of updates occurred on the same day the content was flagged, accounting for 1,784 posts.
  • Over a Week: A smaller number of updates happened after more than a week, accounting for 730 posts.
  • Next Day: Minimal activity was observed for updates the next day (45 posts).
  • Within a Week: A very small group returned to update their content within a week (25 posts).

Key Insight: The bulk of user updates happen immediately on the same day, with a smaller secondary group returning after an extended period of over a week.
